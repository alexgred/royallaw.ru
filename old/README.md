# Royal Law site
Site for Royal Law company.

## Install
Clone repository.
```sh
git clone https://github.com/alexgred/royallaw.git
```
Install gulp-cli global.
```sh
npm i gulp-cli
```
#### For develop
```sh
gulp dev
```
#### For production
```sh
gulp deploy
```
#### Clean derectory from deploy
```sh
gulp clean
```
